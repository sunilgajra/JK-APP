# JK Trade Manager

A lightweight trade management app for managing buyers, suppliers, deals, payments, company settings, and printable trade documents.

## Features

- Buyers management
- Suppliers management
- Deals management
- Payment tracking
- Company settings
- Printable:
  - Proforma Invoice
  - Commercial Invoice
  - Packing List
  - Certificate of Origin
- CSV export
- Supabase backend
- GitHub Pages deployment
